# AI Camera IMX500

## Camera setup on Raspberry Pi 5

The Sony IMX500 is a specialized camera module designed for edge AI applications. It features an integrated AI processing unit that allows for on-device inference, making it ideal for real-time applications without the need for cloud processing.

```bash
# Update and install dependencies
sudo apt install python3-picamera2 -y
sudo apt install imx500-all -y
```

### Convert yolo to imx500 format

Usually we use 

```
yolo export model=../models/<model_name>.pt format=imx
```

It will create a folder with `packerOut.zip` inside.


### Load the model in the IMX Camera
 
Follow the instructions from the [Ultralytics documentation](https://docs.ultralytics.com/integrations/sony-imx500/#software-prerequisites)

Install the modlib library to load the model on the camera and run inference. This library provides a high-level API to interact with the camera and deploy models.

```bash
pip3 install git+https://github.com/SonySemiconductorSolutions/aitrios-rpi-application-module-library.git --break-system-packages
```

Example code

```python
from modlib.apps import Annotator
from modlib.devices import AiCamera
from modlib.models import COLOR_FORMAT, MODEL_TYPE, Model
from modlib.models.post_processors import pp_od_yolo_ultralytics

class YOLO(Model):
    def __init__(self):
        super().__init__(
            # Zip file generated by yolo export (imx format)
            model_file="/path/to/<exported_model>/packerOut.zip", 
            model_type=MODEL_TYPE.CONVERTED,
            color_format=COLOR_FORMAT.RGB,
            preserve_aspect_ratio=False,
        )
        
        # Label loading
        try:
            self.labels = np.genfromtxt(
                "/path/to/<exported_model>/labels.txt",
                dtype=str, delimiter="\n"
            )

    def post_process(self, output_tensors):
        return pp_od_yolo_ultralytics(output_tensors)

device = AiCamera(frame_rate=16) # reduce frame rate to avoid overheating the camera
model = YOLO()
device.deploy(model)
```

